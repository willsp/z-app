# Z-App

A simple webapp to make a list continuously scroll in either direction... no plugins or libraries allowed.

## To Install

Be sure to have (nodejs)[1] and (grunt-cli)[2] installed.

```
    git clone https://github.com/willsp/z-app
    cd z-app
    npm install
    grunt serve
```

[1]: http://nodejs.org/
[2]: http://gruntjs.com/
